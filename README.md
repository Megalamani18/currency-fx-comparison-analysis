# Dynamic Currency FX Comparison & Interactive Cross Rate Analysis

An interactive currency analysis tool built in Excel allowing users to 
compare any of 25+ global currencies against EUR using a dynamic 
spinner-controlled cross rate chart.

## How It Works
- Select any currency using the **spinner input**
- The FX cross rate chart **automatically updates** to show the 
  selected currency trend against EUR
- Adjust the **start and end date** to change the time range
- All 25+ currency pairs update in real time

## What it covers
- 25+ currency pairs tracked against EUR
- Price, High, Low and Daily % Change for each pair
- Dynamic cross rate trend chart — updates on spinner change
- Adjustable date range for historical comparison
- Fully interactive — no formulas need to be changed manually

## Currencies Tracked
- Middle Eastern: KWD, BHD, OMR, JOD, QAR, AED, SAR
- European: GBP, CHF, EUR, BGN
- Americas: USD, CAD, BZD, BBD, KYD, XCD
- Asia Pacific: SGD, BND, AUD, FJD, MYR, CNY
- Other: NZD, AZN, GEL, AWG

## Tools Used
- Microsoft Excel
- Spinner form controls
- Dynamic charting
- FX Market Data
